# Esports and Strategy Games

Use this reference for MOBA, FPS, RTS, card games, auto battlers, fighting games, tactical games, board games, wargames, and MMO team encounters. Keep the analysis inside virtual rules and match mechanics.

## Allowed Tactical Language

It is acceptable to discuss attack, defense, flanking, baiting, encirclement, concentration, attrition, decisive fights, retreats, trades, and tempo when the target is clearly a virtual unit, map area, in-game objective, opponent team in a match, or rules-defined game state.

Do not transfer these terms into real-world harm or coercion against real people.

## Generic Game Analysis

Use the standard game template:

```markdown
## Current Game State
## Principal Contradiction
## Stage Read
## Our Conditions vs Opponent Conditions
## Map/Resource Key Point
## Where to Concentrate Advantage
## Recommended Play
## Risks and Counters
## Next Review Check
## Mao Selected Works Frame
```

Common principal contradictions:

- cannot access vision or information safely
- losing resource trades despite winning fights
- team composition wants a different stage or terrain than current play
- local fights do not convert into objectives
- communication delay prevents concentrated action
- opponent controls tempo and forces reactive decisions

## MOBA Mapping

Stage reads:

- early: lane stability, jungle pathing, vision foundations, first objective setup
- mid: lane assignment, tempo, vision line, pick threat, objective stacking
- late: decisive vision, cooldown tracking, side-lane pressure, fight selection

Framework translations:

- principal contradiction: economy, vision, objective timing, comp scaling, wave state, cooldowns, or coordination
- global/local: kill value must be judged by tower, dragon/baron/roshan, map control, tempo, or recall timing
- concentration: create a numbers, cooldown, item, or vision advantage at one objective
- initiative: push waves, threaten cross-map trade, set vision first, force opponent to answer
- strategic endurance/tactical resolution: stall toward scaling while taking fast fights only where conditions are favorable

## FPS Mapping

Stage reads:

- default/control phase
- execute phase
- retake phase
- economy management phase
- adaptation phase across rounds

Framework translations:

- investigation: review deaths, utility timing, spacing, trade rate, first-contact patterns
- principal contradiction: information deficit, utility mismatch, spacing, weak trade discipline, predictable defaults, economy pressure
- concentration: stack utility and timing onto one lane, bombsite, choke, or retake path
- global/local: opening duel is useful only if it converts into map control, bomb plant, economy damage, or safe reset
- feedback: isolate one round pattern and drill it

## RTS and Strategy Mapping

Stage reads:

- opening build
- scouting and adaptation
- expansion/economy
- timing attack
- tech transition
- late-game control

Framework translations:

- principal contradiction: economy, tech, scouting, production, map control, or army composition
- concentration: hit a timing window before opponent tech or economy pays off
- initiative: scouting forces better decisions; harassment can shift opponent attention
- global/local: a successful raid matters if it changes economy, production, or map control

## Review Prompts

Use these for replay analysis:

- What was the principal contradiction before the losing moment?
- Did the team fight for the global win condition or a local distraction?
- Which stage were we in, and did our play match that stage?
- Where did we try to cover every front instead of concentrating advantage?
- What single recurring pattern should be tested in the next block?
